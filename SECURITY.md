# Security Policy

## Supported Versions
Security fixes are provided for the latest `main` branch state.

## Reporting a Vulnerability
Please do not report security vulnerabilities in public issues.

Use GitHub Security Advisories to report privately:
- https://github.com/tobiaswaelde/ezinventory/security/advisories/new

Please include:
- Vulnerability type and impact
- Affected component(s)
- Reproduction steps / PoC
- Suggested remediation (if available)

## Response Targets
- Initial triage response: within 7 days
- Status updates: as progress is made
- Fix timeline: depends on severity and complexity

## Disclosure
After remediation, we may publish a coordinated disclosure with:
- affected versions/components
- remediation summary
- credit (if desired)
